并行动漫集团
练习1 - 制作一个平行组

分组是重要的，所以我们   上手的运动

在这个练习中，创建一个并行组   动画，拥有3个不同的动画。每个孩子应该运行300像素，   500像素，700像素从顶部分别5秒.

提示：你应该先创建儿童则   包括孩子进组。您可能还需要创建   更多的动画的div在html部分，改变其颜色   取决于你的喜好.

显示解决方案
var A = new Animation(document.querySelector("#a"), {top: "300px"}, 5);
var B = new Animation(document.querySelector("#b"), {top: "500px"}, 5);
var C = new Animation(document.querySelector("#c"), {top: "700px"}, 5);
new AnimationGroup([A, B, C]);
